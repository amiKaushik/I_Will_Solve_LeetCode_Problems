## Comparison of these methods

| Aspect         | Sorting Method          | Priority Queue Method            |
| -------------- | ----------------------- | -------------------------------- |
| Core idea      | Sort once, pick largest | Pick largest each turn           |
| Time           | O(n log n)              | O(n log n)                       |
| Space          | O(1)              | O(n)                             |
| Implementation | Simpler                 | More intuitive greedy simulation |
